# 📦 Sistema de Controle de Estoque em C

Sistema de controle de estoque desenvolvido em linguagem C para gerenciamento básico de produtos.

O projeto permite cadastrar produtos, visualizar informações armazenadas e realizar alterações no estoque através de um menu interativo executado no terminal.

Este projeto foi criado com o objetivo de praticar conceitos fundamentais da programação estruturada, organização de dados e lógica de negócios utilizando a linguagem C.

---

## 🚀 Funcionalidades

### Cadastro de Produtos

Permite registrar novos produtos informando:

- Nome
- Preço
- Quantidade em estoque

Cada produto recebe automaticamente um código identificador único.

### Consulta de Produtos

Exibe todos os produtos cadastrados contendo:

- Nome
- Preço
- Quantidade em estoque
- Código do produto

### Controle de Estoque

Permite atualizar o estoque de um produto utilizando seu código identificador.

É possível:

- Adicionar unidades ao estoque
- Remover unidades do estoque

### Menu Interativo

Sistema executado via terminal com navegação simples através de opções numéricas.

## ▶️ Como Executar

### 1. Clone o repositório

```bash
git clone https://github.com/shtMCe/Controle-de-estoque-basic.git
```

### 2. Acesse a pasta do projeto

```bash
cd Controle-de-estoque-basic
```

### 3. Compile o programa

```bash
gcc main.c -o estoque
```

### 4. Execute

```bash
./estoque
```
